# Modelos da landing 3D

Coloque os modelos finais nesta pasta com estes nomes:

- `obj1.glb` — modelo principal das duas primeiras telas
- `obj2.glb`
- `obj3.glb`
- `obj4.glb`
- `obj-samuel.glb` — boneco do founder Samuel Victor
- `obj-dion.glb` — boneco do founder Dion Vitor

Os arquivos são descobertos automaticamente durante o build. Enquanto um modelo
não existir, a landing usa uma geometria abstrata equivalente como fallback.
